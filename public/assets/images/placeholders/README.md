# Image placeholders

Drop your final optimized visuals here and reference them from components/data.

Recommended replacements (see `docs/higgsfield-prompts.md` for generation prompts):

| File name (suggested)      | Used for                          | Format |
| -------------------------- | --------------------------------- | ------ |
| `hero-loop.webm` / `.mp4`  | Optional hero background video    | webm + mp4 |
| `oil-gas.webp`             | Oil & Gas Measurements section    | webp   |
| `pipeline.webp`            | Pipeline & Asset Integrity        | webp   |
| `fire-safety.webp`         | Fire Safety Systems               | webp   |
| `marine.webp`              | Marine Services                   | webp   |
| `compliance.webp`          | Compliance section background     | webp   |
| `services-bg.webp`         | Services page hero background     | webp   |

The current build ships without photographic imagery — it uses an SVG metrology
gauge, gold lighting, and grid backdrops so the site looks complete and premium
out of the box. Add images progressively; nothing breaks if these are absent.
